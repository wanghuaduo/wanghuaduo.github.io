简单的hexo博客搭建。
新手新手。继续修改中。

node.js安装

搜索http://nodejs.org/进入官网，下载LTS版。然后安装，一直next就行了
打开cmd界面然后输入node -v查看下载的node.js的版本

安装git，并配置好git

注册github账户和新建仓库

安装hexo

并在合适的位置创一个名叫blog的文件夹，用来装生成的文件夹
打开cmd。

输入npm install hexo -g

然后就开始安装hexo

安装成功后可以输入hexo -v检查是否安装成功，顺带查看安装的版本

然后输入hexo init。意思是初始化hexo

接着输入npm install(npm是node里面的一个工具）

然后输入hexo -g

再接着输入 hexo s（表示开始启动）

会出现一个 http://localhost:4000/
点开后会出现

但到现在还没有做完

配置github

将里面的账户信息变成自己的信息

输入cd /.ssh

输入ssh-keygen-t rsa-C”你的邮箱”

回车
最后在c盘生成俩文件夹

登录github

点击Settings

新建一个SSH and GPG Keys

将C盘里的下面的一个文件用记事本打开
将里面的内容复制到key里

打开blog文件夹 然后打开-config文件

要安装 在命令框中输入

然后hexo clean

在输入hexo g

在hexo d

在中间过程中要输入你的名户名及密码

等待 然后会有你的地址生成 在浏览器中打开你的地址就可以看到你所搭建的东西了
